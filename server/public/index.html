Choose a name for your Data Pack
A namespace is used to identify the data and like all file and folder names inside the data folder it should only contain lowercase alphanumerical characters, -, _, and .. It is advised to use a unique namespace to prevent clashes between multiple datapacks. The minecraft namespace should only be used when overwriting default data.
